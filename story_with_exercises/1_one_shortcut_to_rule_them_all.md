# Trick 1: One Shortcut To Rule Them All

No idea in which menu a specific command is?

Use the "Find Action" shortcut!

| Linux + Windows                         | MacOS                                |
|:----------------------------------------|:-------------------------------------|
| <kbd>Ctrl</kbd><kbd>⇧</kbd><kbd>A</kbd> | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>A</kbd> |
